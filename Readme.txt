The Library App is an application for managing books and magazines in the library.

When you launch the app, you will need to select the format in which you will store your data.
You can choose between two options by entering SERIAL or CSV.

Once you have selected the format of your stored data, you can:
- add a new book to the library,
- add a new magazine to the library,
- display available books in the library,
- display available magazines in the library,
- remove a book from the library,
- remove a magazine from the library,
- add a user to the library,
- display library users,
- search for a book by title.
